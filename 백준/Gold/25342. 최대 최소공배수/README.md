# [Gold IV] 최대 최소공배수 - 25342 

[문제 링크](https://www.acmicpc.net/problem/25342) 

### 성능 요약

메모리: 2020 KB, 시간: 4 ms

### 분류

유클리드 호제법, 수학, 정수론

### 문제 설명

<p>$1$부터 $N$까지의 수가 있다. 최소공배수가 최대가 되도록 서로 다른 3개의 수를 선택해 보자.</p>

### 입력 

 <p>첫째 줄에 테스트케이스의 개수 $T$가 주어진다. ($1 \leq T \leq 1\,000$)</p>

<p>둘째 줄부터 $T$개의 줄에 각각 자연수 $N$이 주어진다. ($3\leq N \leq 100\,000$)</p>

### 출력 

 <p>각 테스트케이스마다, 최소공배수의 최댓값을 한 줄에 하나씩 차례대로 출력한다.</p>
