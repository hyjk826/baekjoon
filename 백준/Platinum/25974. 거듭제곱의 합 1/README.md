### 문제 설명

<p>$1$부터 $n$까지 모든 자연수의 $p$ 거듭제곱의 합을 $10^9+7$로 나눈 나머지를 구하시오. 다시 말해,</p>

<p>$$\left(\sum_{k=1}^{n}k^p\right)\mod\left(10^9+7\right)$$</p>

<p>을 구하시오.</p>

### 입력 

 <p>첫 줄에 $n$과 $p$가 공백을 사이에 두고 주어진다.</p>

### 출력 

 <p>거듭제곱의 합을 $10^9+7$로 나눈 나머지를 출력한다.</p>
